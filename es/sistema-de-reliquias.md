# 🔮 Sistema de Reliquias

Las reliquias son potentes objetos pasivos que se equipan en un **contenedor dedicado**, independiente de tu inventario habitual. Cada reliquia otorga bonificaciones permanentes a tus atributos.

![](../assets/custom/relics.png)

{% hint style="info" %}
**Reglas de los Casilleros de Reliquia:**
- **Casilleros Iniciales:** Todos los jugadores comienzan con **2 casilleros de reliquia**, ampliables hasta un máximo de **4 casilleros**.
- **Límite de Unicidad:** **No puedes equipar dos reliquias del mismo tipo** simultáneamente.
{% endhint %}

## 📜 Expansión de Casilleros

Para desbloquear casilleros adicionales para tus reliquias:

- **Pergamino de Casillero de Reliquia (Relic Slot Scroll):** Desbloquea de forma permanente **+1 casillero de reliquia** en tu personaje.
- **Límite Máximo:** El límite máximo es de **4 casilleros**. Los pergaminos no se pueden utilizar más allá de este límite.

![](../assets/custom/relic_scroll.png)
