# 🩸 Sistema de Sacrifício

O Sistema de Sacrifício permite que os jogadores contribuam cooperativamente com Eternum Coins para ativar bônus globais (boosts) em todo o servidor. Quando uma contribuição é feita, uma janela de tempo limitada se abre para que o servidor inteiro colabore e melhore o nível final do bônus.

{% hint style="info" %}
Todos os sacrifícios exigem **Eternum Coins**. O sistema retira moedas automaticamente tanto do seu inventário quanto do saldo da sua conta na Loja.
{% endhint %}

## 🪙 Moeda
Todos os sacrifícios exigem **Eternum Coins**. O sistema automaticamente retira as moedas tanto do seu inventário quanto do saldo da sua conta na Loja.

## 🏆 Categorias e Recompensas
Existem quatro categorias globais nas quais os jogadores podem contribuir, cada uma oferecendo um **bônus de 1 hora** para o servidor:

1. **Experiência (Experience)**: Aumenta a taxa de experiência global.
2. **Loot**: Aumenta a taxa de loot global.
3. **Skill e Magic Level**: Aumenta a taxa global de progressão de habilidades e nível mágico.
4. **Chance de Monstros Elite**: Aumenta a chance de monstros nascerem como Elites.

## ⚙️ Como Funciona
Os jogadores doam Eternum Coins para a categoria que escolherem. Assim que a primeira doação for feita, uma contagem regressiva se inicia. O servidor deve colaborar para doar o máximo de moedas possível antes que o tempo acabe. 

O poder final do bônus de 1 hora é baseado na quantidade total de moedas doadas por todos os jogadores. Se os jogadores atingirem a meta máxima, a versão mais forte do bônus será ativada imediatamente! Caso o servidor não atinja a quantidade mínima, o sacrifício falha e as moedas são perdidas.

{% hint style="warning" %}
Um bônus ativado com sucesso sempre dura exatamente **1 hora**. Não é possível contribuir para uma categoria se o bônus dela já estiver ativo.
{% endhint %}
