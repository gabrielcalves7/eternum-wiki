# Tibia Eternum Wiki

Source Markdown for the public Tibia Eternum GitBook wiki.

GitBook Sync should connect this repository to the English and Portuguese GitBook spaces. Staff may author permanent guides in GitBook; GitBook Sync mirrors those edits here. The Laravel website will generate only the spell references.

## Structure

- `en/` — English content
- `pt/` — Portuguese content

The generated spell pages are owned by the website exporter. Do not edit their generated bodies manually.
