# 🩸 System Poświęceń

System Poświęceń pozwala graczom na wspólne wpłacanie Eternum Coins w celu aktywowania globalnych wzmocnień (boostów) dla całego serwera. Po dokonaniu wpłaty otwiera się ograniczone czasowo okno, w którym cały serwer może wpłacać monety i podnosić poziom ostatecznego wzmocnienia.

{% hint style="info" %}
Wszystkie poświęcenia wymagają **Eternum Coins**. System inteligentnie pobiera monety zarówno z Twojego ekwipunku, jak i ze stanu konta w Sklepie (Shop Balance).
{% endhint %}

## 🪙 Waluta
Wszystkie poświęcenia wymagają **Eternum Coins**. System inteligentnie pobiera monety zarówno z Twojego ekwipunku, jak i ze stanu konta w Sklepie (Shop Balance).

## 🏆 Kategorie i Nagrody
Istnieją cztery globalne pule, do których gracze mogą wpłacać monety, z których każda zapewnia inny **1-godzinny boost dla serwera**:

1. **Doświadczenie (Experience)**: Zwiększa globalny mnożnik doświadczenia.
2. **Łup (Loot)**: Zwiększa globalny mnożnik łupu.
3. **Umiejętności i Magia (Skill & Magic)**: Zwiększa globalny mnożnik przyrostu umiejętności i poziomu magii.
4. **Szansa na Elitarne Potwory (Elite Monster Chance)**: Zwiększa szansę na pojawianie się potworów jako Elitarne Potwory.

## ⚙️ Jak To Działa
Gracze przekazują Eternum Coins na wybraną przez siebie kategorię. Gdy zostanie dokonana pierwsza wpłata, rozpoczyna się odliczanie. Serwer musi współpracować, aby wpłacić jak najwięcej monet, zanim upłynie czas. 

Ostateczna moc 1-godzinnego boosta zależy od łącznej liczby monet wpłaconych przez wszystkich graczy. Jeśli graczom uda się osiągnąć maksymalny cel, najsilniejsza wersja wzmocnienia aktywuje się natychmiast! Jeśli serwerowi nie uda się osiągnąć minimalnego wymogu, poświęcenie kończy się niepowodzeniem, a monety zostają utracone.

{% hint style="warning" %}
Pomyślnie aktywowane wzmocnienie zawsze trwa dokładnie **1 godzinę**. Nie możesz wpłacać monet do puli, której boost jest już aktywny.
{% endhint %}
