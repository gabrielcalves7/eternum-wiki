# 🔮 Sistema de Relíquias

Relíquias são itens passivos poderosos equipados em um **recipiente dedicado** separado de seu inventário regular. Cada relíquia concede bônus de atributos permanentes.

![](../assets/custom/relics.png)

{% hint style="info" %}
**Regras de Slots de Relíquias:**
- **Slots Iniciais:** Todo jogador começa com **2 slots de relíquia**, expansíveis até **4 slots**.
- **Limite de Unicidade:** Você **não pode equipar duas relíquias do mesmo tipo** simultaneamente.
{% endhint %}

## 📜 Expansão de Slots

Para desbloquear slots adicionais para suas relíquias:

- **Relic Slot Scroll (Pergaminho de Slot de Relíquia):** Desbloqueia permanentemente **+1 slot de relíquia** em seu personagem.
- **Limite Máximo:** O limite máximo é de **4 slots**. Pergaminhos não podem ser usados além deste limite.

![](../assets/custom/relic_scroll.png)
