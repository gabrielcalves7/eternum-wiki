# 🛒 Sistema de Mercado

Compra y vende objetos con facilidad utilizando el Mercado dentro del juego.

![](../assets/custom/market.png)

{% hint style="info" %}
El **Sistema de Mercado** permite a los jugadores crear ofertas para comprar o vender objetos directamente dentro del cliente.
{% endhint %}

## 🏷️ Funciones

- **Crear Ofertas:** Establece tu precio y cantidad para comprar o vender.
- **Manejo Avanzado de Objetos:** Compatible totalmente con objetos con **Rareza** y otros atributos personalizados.
- **Oferta y Demanda:** Consulta las publicaciones actuales para evaluar el valor de mercado de tu botín.
