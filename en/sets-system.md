# Sets System

The Sets System provides players with powerful stat bonuses when equipping multiple items belonging to the same equipment set.

## 1. How Sets Work
Items in the game can belong to specific sets (e.g., the "Demon" set). When you equip multiple pieces of the same set, you unlock special bonuses. 

## 2. Set Bonuses
Set bonuses are based on the number of pieces you have equipped from that set. 

### Cumulative Bonuses
Bonuses are **cumulative**. For example, if you equip 3 pieces of a set, you will receive the bonuses for having 2 pieces *plus* the bonuses for having 3 pieces combined.

### Types of Bonuses
Bonuses can include various stat improvements such as:
- Increased Maximum Health and Mana
- Enhanced Damage or Magic Level
- Increased Critical Hit Chance or specific element strike chances
- Condition Immunities (e.g., paralyze immunity)
- Increased specific weapon skills (Sword, Axe, Club, Shielding)
