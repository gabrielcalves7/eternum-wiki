# 💰 System Autoloot

System **Autoloot** pozwala graczom na automatyczne zbieranie określonych przedmiotów z ciał potworów. Został zaprojektowany z myślą o maksymalnej wygodzie, umożliwiając płynne zarządzanie listą łupów.

## ⚙️ Funkcje i Mechaniki

### 🎒 Podstawowy Autoloot
- **Pojemność podstawowa:** Rozpoczynasz z **8 slotami** na przedmioty na swojej liście autoloota.
- **Mechanika:** Gdy otwierasz ciało potwora, system automatycznie wyciąga przedmioty pasujące do Twojej listy i umieszcza je w Twoim ekwipunku.

### ⚡ Łupienie Bez Klikania (Loot Without Click)
- **Mechanika:** Gracze mogą odblokować zdolność **Łupienie Bez Klikania** za pomocą specjalnego przedmiotu.
- **Efekt:** Po odblokowaniu przedmioty z Twojej listy trafiają bezpośrednio do Twojego ekwipunku w momencie śmierci potwora, eliminuje to potrzebę fizycznej interakcji z ciałem.

### 🪙 Monety Bankowe (Pozwolenie Midasa)
{% hint style="info" %}
**Bezpośredni Depozyt Bankowy:** Użycie **Midas Permit** (Pozwolenia Midasa) trwale przyznaje dostęp do funkcji Bank Coins. Gdy funkcja jest aktywna, wszelkie zdobyte monety złota, platyny lub kryształu omijają Twój ekwipunek i są wpłacane bezpośrednio na Twoje konto bankowe.
{% endhint %}

### 📈 Zwiększenie Pojemności
- **Mechanika:** Gracze mogą trwale zwiększyć maksymalną liczbę slotów autoloota za pomocą specjalnego przedmiotu zwiększającego pojemność.

## 🖥️ Zarządzanie Interfejsem

Możesz łatwo zarządzać całą konfiguracją autoloota, w tym dodawać lub usuwać przedmioty, bezpośrednio z poziomu interfejsu klienta.

## 🗂️ Mapowanie Pojemników (Kategoryzacja Łupu)

Możesz organizować swoje łupy, kierując określone kategorie przedmiotów do wyznaczonych pojemników (takich jak konkretne plecaki) w swoim ekwipunku za pomocą interfejsu klienta.

### Dostępne Kategorie

| Kategoria | Opis |
| --- | --- |
| **Ekwipunek** | Bronie, pancerze, nogawice, hełmy, tarcze, buty, pierścienie, amulety |
| **Wartościowe przedmioty** | Kamienie szlachetne, perły, rzadkie produkty z potworów |
| **Mikstury** | Mikstury zdrowia i many, płyny |
| **Runy** | Magiczne runy i zwoje czarów |
| **Złoto** | Złote, platynowe i kryształowe monety |
| **Amunicja** | Strzały, bełty, włócznie, gwiazdki do rzucania |
| **Różne** | Przedmioty z zadań, jedzenie, klucze, narzędzia |
| **Domyślne** | Niekategoryzowane przedmioty |
