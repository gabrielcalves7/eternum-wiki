# 🛒 Sistema de Mercado

Compre e venda itens com facilidade usando o Mercado do jogo.

![](../assets/custom/market.png)

{% hint style="info" %}
O **Sistema de Mercado** permite que os jogadores criem ofertas de compra ou venda de itens diretamente no client.
{% endhint %}

## 🏷️ Funcionalidades

- **Criar Ofertas:** Defina o preço e a quantidade para compra ou venda.
- **Manuseio Avançado de Itens:** Suporta totalmente itens com **Raridade** e outros atributos customizados.
- **Oferta e Demanda:** Verifique as listagens atuais para avaliar o valor de mercado do seu loot.
